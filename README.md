**Симулятор ПДР (MVS Simulator) 🚗**

Інтерактивний додаток для підготовки до іспитів на отримання водійського посвідчення. Проєкт створений з фокусом на зручний користувацький досвід (UX) та ефективне засвоєння матеріалу.

**🎯 Про проєкт**

Цей симулятор дозволяє користувачам проходити тестування зі знання Правил дорожнього руху. База питань структурована у форматі JSON (`questions_db.json`), що дозволяє зручно оновлювати контент та збирати аналітику відповідей для покращення продукту.
Додаток розроблявся з комплексним продуктовим підходом: від ідеї до повноцінного релізу на різних платформах, щоб закрити потреби максимально широкої аудиторії.

**🚀 Як завантажити та спробувати**

Готові версії програми вже зібрані та доступні в розділі **Releases** (у меню справа).
Виберіть зручний для вас формат:

* **Windows:** завантажте файл `mvs_simulator.exe`
* **Android:** завантажте додаток `mvs_simulator.apk`
* **Web-версія:** завантажте `mvs_website.zip` (або просто відкрийте `index.html` з цього репозиторію).

## 🛠 Стек технологій

Проєкт побудований на базових вебтехнологіях, що забезпечує його легкість, кросплатформність та швидку роботу:

* **JavaScript (Vanilla JS)** — основна логіка симулятора, обробка дій користувача та підрахунок результатів.
* **HTML5 / CSS3** — структура та адаптивний інтерфейс.
* **JSON** — база даних питань та відповідей.

## 💡 Основні компетенції, застосовані в проєкті

* **Product Management:** Проведення проєкту через повний життєвий цикл — концепція, розробка, пакування та реліз на різних операційних системах.
* **Data & Analytics:** Логічна структуризація бази даних для тестування, закладена основа для аналізу поведінки та успішності користувачів.
* **Технічна реалізація:** Самостійне написання клієнтської частини на JavaScript, розуміння архітектури вебдодатків та роботи інтерфейсів під капотом.
